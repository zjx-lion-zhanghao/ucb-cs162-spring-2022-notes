This repo contains my personal notes, reading summaries, debugging logs, and learning reflections while studying CS162 Spring 2022.

The main goal is to build a systematic understanding of operating systems, including:

- System calls

- Virtual memory
- File systems
- Distributed systems
- Pintos project design ideas
